# دليل الإعداد - OtakuZone

## المتطلبات الأساسية

- Node.js 18+ 
- npm أو yarn
- Git

## خطوات الإعداد

### 1. تثبيت المتطلبات

```bash
npm install
```

### 2. إعداد قاعدة البيانات

قم بإنشاء ملف `.env` في المجلد الرئيسي:

```env
DATABASE_URL="file:./dev.db"
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="your-secret-key-here-change-in-production"
JIKAN_API_URL="https://api.jikan.moe/v4"
```

### 3. إنشاء قاعدة البيانات

```bash
npm run db:generate
npm run db:push
```

### 4. ملء قاعدة البيانات بالبيانات الأولية

```bash
npm run db:seed
```

سيتم إنشاء:
- 8 تصنيفات (شونين، شوجو، إيسيكاي، إلخ)
- مستخدم إداري (admin@otakuzone.com / admin123)
- مستخدم تجريبي (user@test.com / user123)
- مواضيع تجريبية

### 5. تشغيل المشروع

```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.
